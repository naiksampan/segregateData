# segregateData
Collection of INDIAN license plates . It should be in HD quality. Collect it from different sources , movies etc. 2. Crop the license plate first . After that, crop the alphabets and numbers present in it. 3. Convert the background into white color . The digits or alphabets should be in black as an example  shown in the meeting. 4. Resize them into 28x28 pixel size 5. Lastly make different folders starting from 0 to 9 and then AtoZ. Segregate all the letters and alphabets properly and then rename accordingly.
